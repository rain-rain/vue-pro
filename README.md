# vue 框架试水——H5项目

#安装依赖包
```
npm install
```

#启动
```
npm run dev
```
### 本地访问
http://127.0.0.1:8888/


#编译项目
```
npm run build
```


### 目录结构

```
├─vue-pro
  ├─build             node操作
  ├─config            配置文件
  ├─mock              本地mock文件
  ├─node_modules      node依赖包，包含node组件和前端组件，不需要提交到git
  ├─src               项目源码目录
    ├─asset           静态资源
    ├─components      vue组件
    ├─images          图片资源
    ├─pages           页面
    ├─router          路由
    ├─scripts
      ├─config        共用js配置
      ├─libs          js库文件
      ├─utils         工具类
      ├─boot.js       H5适配
    ├─App.vue
    ├─main.js
  ├─index.html
  ├─package.json      项目信息


```
